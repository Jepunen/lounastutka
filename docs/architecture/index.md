# Architecture

The description of the architecture of the system. 


The diagrams are built using PlantUML with some diagrams utilizing the C4 model extension available here: https://github.com/plantuml-stdlib/C4-PlantUML#deployment-diagram

If and when changes are made to the diagrams (and when the architecture changes), they can be recreated under `docs/architecture/imgs` with:

```bash
[project_root]$ cd docs/architecture/
[architecture]$ plantuml diag_src/*.puml -o ../imgs
```

**NOTE**
    You need to have 'plantuml' installed, 
    the C4 model extension should be fetched 
    automatically from the github repository.
